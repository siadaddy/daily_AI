# 용어집

| 용어 | 뜻 |
|---|---|
| 탭 (tab) | `?tab=` 쿼리로 전환되는 5개 화면 단위: `newsletter`, `reports`, `music`, `office`, `portfolio`. [[00-src-개요]] 참고 |
| 카드뉴스 (card news) | AI가 매일 생성하는 5개의 짧은 뉴스 카드. [[card_news]] 테이블에 저장, [[news_cards]](원본 뉴스)와 혼동 주의 |
| 원본 뉴스 (raw news) | 네이버 뉴스 API로 수집한 가공 전 뉴스. [[news_cards]] 테이블 |
| 아티클 (article) | AI 편집장이 매일 쓰는 블로그 글 1건. [[articles]] 테이블 |
| period_type | [[weekly_reports]] 테이블에서 `weekly`/`monthly`를 구분하는 판별 컬럼 — 주간과 월간 리포트가 한 테이블에 함께 저장됨을 나타내는 핵심 개념 |
| AI Office | [[office]] 탭 — pipeline 에이전트들의 실시간 활동을 시각화하는 화면. [[agents-logs]] 테이블을 Realtime 구독 |
| 에이전트 (agent) | pipeline이 실행하는 개별 AI 역할: 수집봇, 박기획(planner), 이작가(writer), 최디자(designer), AI주간트렌드(weekly_trend), 한뮤직(music_curator). [[pipeline-agents]] 참고 |
| Music Universe | [[music]] 탭 — Three.js 3D 음악 시각화. iframe으로 격리된 정적 앱, `public/music/*.json`을 [[ai-crew-개요|ai-crew]]가 생성 |
| service-role 클라이언트 | RLS(Row Level Security)를 우회하는 Supabase 관리자 클라이언트. `createAdminClient()`, 서버 전용 — [[라이브러리-및-상태관리]] 참고 |
| MOC (Map of Content) | 이 볼트의 전체 목차 역할을 하는 노트. [[00-지도]] |
