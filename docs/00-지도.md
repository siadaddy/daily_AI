# 지도 (Map of Content)

이 볼트는 한국 AI 뉴스 대시보드 프로젝트의 컴포넌트/API/파이프라인/데이터베이스를 모듈 단위로 문서화한다.
전체 명령어, 환경변수, 스타일링 규칙 등 "빠르게 참고할 사실"은 여기서 반복하지 않고 저장소 루트의
`CLAUDE.md`를 참고한다. 이 볼트는 그보다 한 단계 더 들어간 "왜/어떻게" 설명과 상호 링크를 제공한다.

## 이 문서를 최신 상태로 유지하는 법

각 노트는 frontmatter에 `source_paths`(이 노트가 다루는 실제 코드 파일 목록)를 갖고 있다.
코드를 수정한 뒤에는 아래처럼 grep해서 확인해야 할 노트를 찾는다.

```bash
grep -rl "src/app/api/reports/generate" docs/
```

`status: 초안` 은 아직 코드로 재확인하지 않았거나 추측이 섞인 내용, `status: 확인됨`은
실제 코드/DB를 보고 검증한 내용을 의미한다. 노트를 수정했다면 `last_reviewed` 날짜도 갱신한다.
